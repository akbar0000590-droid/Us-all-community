Accessibility Statement

Last Updated: March 15, 2025

Our Commitment

The US-All Community Trust Relief Program is committed to ensuring digital accessibility for all people, regardless of ability. We are continuously working to improve the user experience and apply relevant accessibility standards.

♿

WCAG 2.1 AA

We strive to meet or exceed WCAG 2.1 Level AA standards

🔊

Screen Readers

Compatible with JAWS, NVDA, and VoiceOver

⌨️

Keyboard Navigation

Full keyboard accessibility throughout

Accessibility Features

  • Keyboard Navigation: All functionality is available using a keyboard (Tab, Enter, Space, Arrow keys)
  • Screen Reader Support: Proper heading structure, ARIA labels, and alt text for images
  • Color Contrast: Text meets WCAG contrast requirements (minimum 4.5:1 for normal text)
  • Resizable Text: Text can be enlarged up to 200% without loss of functionality
  • Focus Indicators: Clear visual focus indicators for keyboard users
  • Skip Links: "Skip to main content" links available on all pages

Document Accessibility

We provide accessible document formats upon request:

  • HTML versions of all policies
  • Large print documents
  • Braille (upon request)
  • Audio recordings
  • Plain text versions

Ongoing Efforts

We regularly test our website with:

  • Automated accessibility testing tools (WAVE, axe)
  • Manual keyboard-only testing
  • Screen reader testing (NVDA, VoiceOver)
  • User testing with people with disabilities

Known Limitations

While we strive for full accessibility, some areas may need improvement:

  • Older PDF documents (we can provide accessible versions upon request)
  • Third-party content (Facebook, Google Maps) - these platforms have their own accessibility features
